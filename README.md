## Setup
```
$ git clone https://lab.ssafy.com/s05-webmobile3-sub3/S05P13A109.git
$ cd S05P13A109
$ npm install

// backend
$ cd ./S05P13A109/backend/src
$ npm install
$ npx nodemon

// frontend
$ cd ./S05P13A109/frontend
$ npm install
$ npm run serve
```

## Technical Stack
### Backend
- Node.js
- Express
- JWT (JSON Web Tokens)
- MySQL
- sequelize
- Bcrypt

### Frontend
- Vue.js
- Element-ui
